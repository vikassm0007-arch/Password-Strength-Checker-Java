# 🔐 Password Strength Checker (Java)

A simple yet effective Java-based application that evaluates the strength of a user-entered password based on security best practices.

---

## 📌 Description

This project analyzes a password using multiple criteria such as length, character diversity, and complexity. It provides a strength rating (Weak, Medium, Strong) along with actionable feedback to help users create more secure passwords.

---

## ⚙️ Features

- ✔ Checks minimum password length
- ✔ Detects uppercase and lowercase characters
- ✔ Identifies numeric digits
- ✔ Validates presence of special characters
- ✔ Provides strength classification:
  - Weak ❌
  - Medium ⚠️
  - Strong ✅
- ✔ Gives improvement suggestions

---

## 🛠️ Technologies Used

- Java
- Scanner Class (User Input)
- String Manipulation
- Conditional Statements
